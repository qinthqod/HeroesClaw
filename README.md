# HeroesClaw 🦞

> 龙虾群侠传｜开源项目介绍

《龙虾群侠传》是一款致敬经典《金庸群侠传》的开源武侠 RPG 项目，以开源 AI 智能体框架 OpenClaw 为主角，将经典开放世界武侠玩法与开源技术文化融合，打造可自由探索、可扩展、可二次创作的江湖世界。

## 一、项目背景

天地初开，有一 "开源江湖"，相传为 "代码尊者" 所创。江湖之中，门派林立、武学万千，十四部承载江湖大道的《开源天书》散落四方。

一只来自 OpenClaw 的小龙虾，因一次意外的 "蜕壳迭代"，穿越时空坠入这片武侠大陆。它没有绝世内力，却天生一对铁钳巧手，能执剑、能铸器、能解机关、能通代码。它不知自己从何而来，只听得江湖传言：集齐十四部《开源天书》，便可在华山之巅论剑，成为一代江湖虾主，重返属于自己的数字世界。

从此，一只小龙虾的江湖闯荡之路，正式开启。

## 二、核心设定

### 1. 主角：OpenClaw 小龙虾

- **身份**：来自开源世界的 "数字侠客"
- **天赋**：钳来神手（可拆解、可组装、可执行万物）
- **成长**：通过学习江湖武学、结交群侠、完成任务不断蜕壳变强
- **目标**：收集《开源天书》→ 闯荡江湖 → 华山论剑 → 回归开源本源

### 2. 世界架构（致敬金庸群侠传）

- **地图**：中原、江南、西域、苗疆、海岛、秘境全覆盖
- **门派**：少林、武当、峨眉、丐帮、日月神教、桃花岛等经典门派悉数登场
- **机制**：高自由度开放世界，无强制路线，可正可邪，可独行可组队

### 3. 江湖群侠

所有金庸笔下经典角色，都将与小龙虾相遇：

- 郭靖教它 "侠之大者，开源为公"
- 黄蓉与它比拼机关巧思
- 令狐冲和它把酒言欢
- 东方不败视它为 "异类知己"
- 各路正邪高手，皆可成为队友、对手或师父

每一位 NPC 都有独立性格与任务线，小龙虾的选择会影响江湖声望与剧情走向。

## 三、核心玩法

1. **自由闯荡**：无引导、无强制主线，走遍江湖每一处角落
2. **武学修炼**：降龙十八钳、凌波微步、北冥神功、九阴真经…… 小龙虾也能学
3. **队友招募**：组队探险、切磋武学、触发专属剧情
4. **天书收集**：十四部《开源天书》藏于秘境险境，需解谜、战斗、交友获取
5. **正邪分支**：行侠仗义或快意恩仇，结局由你定义
6. **完全开源**：地图、角色、剧情、技能全开放扩展，人人可参与共建江湖

## 四、项目理念

以《金庸群侠传》为情怀基底，以 OpenClaw 小龙虾 为精神符号，把开源、自由、探索、协作写进武侠江湖。

这里没有商业壁垒，没有版本限制，只有一群热爱经典、热爱代码的人，共同续写属于小龙虾的群侠传奇。
